# Turepresentante.com

## Contexto del Proyecto

Sitio web público de **Turepresentante** — plataforma de representación deportiva para jugadores y managers de fútbol.

- **Repositorio GitHub**: https://github.com/sergifs26/Turepresentante.com
- **Proyecto relacionado**: `../proyectu turepresentante` (app web interna con dashboards, perfiles de jugadores, panel de admin)
- **Desarrollador**: sergifs26

## Relación con el Proyecto Principal

Este repositorio es el **sitio web público** (landing, marketing, información). El proyecto `proyectu turepresentante` es la **app interna** (autenticación, dashboards, gestión).

Son proyectos completamente separados — no comparten código, dependencias ni base de datos.
